A beadott anyag tartalma
* build: a release készítését végző ant projekt
* fs-common: JPA entitások
* fs-schemagen: DB séma létrehozását végző program
* fs-web: GWT alkalmazás + service réteg
* lib: jar fájlokat tartalmazó segédprojekt
* lib-common: mindenki számára fordítási időben szükséges jar fájlokat exportáló segédprojekt
* lib-web: web modul számára fordítási időben szükséges jar fájlokat exportáló segédprojekt

Felhasznált library-k (fordításhoz szükséges)
* Commons logging: http://commons.apache.org/logging/
* GWT (2.2.0): http://code.google.com/webtoolkit/
* Commons file upload (1.2.1): http://commons.apache.org/fileupload/
* Java EE 5 API: http://www.oracle.com/technetwork/java/javaee/overview/index.html
